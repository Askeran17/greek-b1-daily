## About the Project

Greek B1 Daily is an interactive language learning application designed to help users practice and improve their Greek language skills at the B1 level. The app generates daily quizzes and exercises using AI-powered content generation, providing a personalized learning experience.

### Features

- **AI-Generated Questions**: Utilizes Google's Gemini AI to create dynamic and contextually relevant Greek language exercises
- **Interactive Quiz Interface**: Clean and intuitive user interface built with React
- **Real-time Feedback**: Instant results and explanations for better understanding
- **Progress Tracking**: Track your learning journey with detailed results
- **Responsive Design**: Works seamlessly on desktop and mobile devices

### Technology Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **AI Integration**: Google Generative AI (Gemini)
- **UI Components**: Lucide React for icons
- **Deployment**: Netlify

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Askeran17/greek-b1-daily.git

# Navigate to the project directory
cd greek-b1-daily

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Building for Production

```bash
npm run build
```
